Raw Shellcode

Paste your raw shellcode bytes here (e.g., from generic Metasploit output). The builder will encrypt it.

Injection Target

  • Remote: Injects into a running `notepad.exe` process (classic Process Injection).
  • Local: Executes within the malware's own process memory (Self-Injection).

Bypass Technique

  • Indirect Syscalls (Hell's Gate): Dynamically resolves SSNs to bypass user-land hooks without hardcoding values.
  • Full Unhooking: Reloads a clean `ntdll.dll` from disk to overwrite any EDR hooks in memory.

Fake Exports (DLL)

Goal: Blend in with legitimate system DLLs (OpSec).

Do NOT use names like bypass or payload. Use standard Microsoft exports:

  • DllRegisterServer (Common in COM DLLs)
  • DllInstall
  • Debug
  • Run
  • UpdateCheck
